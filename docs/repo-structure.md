# Repository structure

The repository has the following layout to help organize the code and environment.

```
thaiflux/
├── .env.example
├── .github/
├── backend/
├── docs/
├── frontend-mobile/
├── frontend-web/
├── infra/
└── scripts/
```

## .env.example
Environment variables example file. Copy this file to .env and fill in the values.

## .github
Folder with GitHub actions workflows.

## backend
Java/Kotlin backend application using Spring Boot. This folder contains the microservices that make up the backend.

## docs
Folder with documentation.

## frontend-mobile
Flutter mobile application.

## frontend-web
React web application.

## infra
Folder with infrastructure as code and configuration.

## scripts
Folder with scripts for automated tasks.
