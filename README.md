# ThaiFlux

[ThaiFlux](https://thaiflux.dev) is the official website of the ThaiFlux application, deployed by [@GregTheRick](https://github.com/GregTheRick).

ThaiFlux aims to be a platform for Thai language learning and practice.

## Roadmap:

### MVP:
- [ ] User registration and login.
- [ ] Practice Thai Scripture (vowels and consonants) with their thai names and corresponding location on a Thai keyboard layout.
